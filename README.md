# Project Name: Keen Keeper

A personal relationship management app that helps you nurture and maintain meaningful connections with the people who matter most in your life.

## Description

Keen Keeper is a web application designed to help you organize, track, and nurture your meaningful relationships. Never lose touch with important people in your life - keep all your friendships on track with intuitive tracking, timeline management, and insightful statistics.

## Technologies Used

- **React 19** - Modern JavaScript UI library
- **Vite** - Lightning-fast frontend build tool
- **React Router 7** - Client-side routing and navigation
- **Tailwind CSS 4** - Utility-first CSS framework
- **DaisyUI** - Component library built on Tailwind
- **Recharts** - Beautiful data visualization library
- **React Icons** - Icon library with 6000+ icons
- **React Toastify** - Toast notifications
- **Context API** - State management for timelines

## Key Features

### 1. **Friend Contact Management**
Browse and manage all your meaningful connections in one place. View friend profiles with contact information, relationship tags (university, best friend, gym buddy, etc.), bio, and interaction goals.

### 2. **Contact Timeline**
Track all your interactions with friends. Record when you last contacted someone and keep a timeline of your meaningful moments together. Monitor if you're on track or overdue for reaching out.

### 3. **Relationship Statistics**
Get insights into your relationships with interactive stats dashboard showing:
- Total friends
- Friends on track
- Friends needing attention
- Monthly interactions
